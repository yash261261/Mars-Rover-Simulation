# Swarm Robots Communication Server
REST API server used as means to exchange information for [Swarm Robot Project][1]


####Instructions:
1. clone this git repository
2. install nodejs
3. run `npm install`
4. run `node app.js`
5. Open browser and check http://localhost:3000/
